## Use on Overleaf
> **Setup note:** This project is intended to be compiled on **Overleaf** and with **GitHub Actions**.
### Option A: Upload a ZIP
1. Click **Code ‚Üí Download ZIP** on GitHub.
2. In Overleaf, create a new project and **Upload Project** (ZIP).
3. Set the **Main file** to `dissertation.tex`.

### Option B: GitHub‚ÄìOverleaf sync
If you use Overleaf‚Äôs GitHub integration, link this repository to your Overleaf project and keep the repo as the single source of truth.
